# 🌌 Exo Planet with Python Turtle

## 📖 Overview
This project was created for fun using **Python’s Turtle graphics library**.  
It generates an abstract spiral pattern resembling an **exoplanet**.  
The program demonstrates how to use Turtle for creative coding, combining loops, geometry, and artistic visuals.

---

## ✨ Features
- 🌀 **Spiral Pattern**: Creates a dynamic spiral using incremental forward and right turns.
- 🌌 **Exoplanet Theme**: Blue pen color on a black background for a cosmic effect.
- 🐢 **Turtle Graphics**: Demonstrates pen movement, screen setup, and continuous drawing logic.
